API Config Builder

为 CN-NewsTTS Bench 生成本地 TTS API 配置 JSON。

这个页面只在本地浏览器中运行，不会上传任何信息，也不会使用 localStorage 保存密钥。导出的文件建议命名为 tts_api_config.local.json，放在 configs/ 或其他本地安全目录，千万不要提交到 GitHub。
最少要填：启用 provider 后，确认 Display Name、Model、Voice，并至少提供一种认证方式：API Key Env Var、API Key 或 Credential File Path。
可以先不填：Sample Rate、Audio Format、Region、Endpoint、Extra JSON。只有对应服务文档明确要求时再补。
榜单约束：页面底部 policy 固定为 raw text only，不允许外部前端、LLM 改写、SSML 或人工修正。
